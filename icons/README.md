# Icons

Place two PNG files here:

- `icon-192.png` — 192×192 px (required for Android add-to-home-screen)
- `icon-512.png` — 512×512 px (required for PWA splash screens)

The icon should feature the SWAT4HCLS logo or the conference acronym on a dark blue background (#0d47a1).

Tools for generating PWA icons from an SVG or PNG:
- https://maskable.app/ — test/generate maskable icons
- https://realfavicongenerator.net/ — full favicon + icon set generator
- https://pwa-asset-generator.netlify.app/ — CLI-based PWA icon generator
